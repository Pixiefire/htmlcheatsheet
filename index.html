HTML Cheat Sheet

HTML

There are many different steps in html that we have to know. Building even a basic webpage can be daunting. The following information is to help us remember those difficult steps, and make future pages easier to build.

Starting/Header

You need the following to begin:

  • What type of HTML are you using?
  • What is the title?
  • What is the CSS link, if you are using one

Type of HTML

This used to distinguish between newer and older html types. The one we are using is !DOCTYPE html.

Title

This is what will show up on the tab in the browser.

CSS Links

Linking a CSS page will greatly improve the appearance of the website. This can be done by doing the following:<link rel="stylesheet" href="styles.css"> or the correct href path on your text editor.

Body

The body of the website consists of many different elements and sections. Paragraphs, tables and images. Paragraphs are the easiest, as they are denoted by a <p></p>. But tables and images take a little bit more work.

Table

When putting a table on a website you should use the following:

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Tables can get confusing, best to list out all tags needed the fill in the middle, that way you don't forget ending tags etc.
